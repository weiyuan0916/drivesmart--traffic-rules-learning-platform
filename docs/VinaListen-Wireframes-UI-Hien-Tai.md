# VinaListen — Text Wireframes (UI Hiện Tại)
## Desktop · Tablet · Mobile

**Dựa trên:** Source code thực tế từ `src/components/listening/`  
**Views:** 8 trang (Overview, Topics, Topic Detail, Practice, Progress, Leaderboard, Bookmarks, History)  
**Navigation:** State-based (không có URL routing), bottom nav trên mobile, sidebar nav trên desktop

## QUY ƯỚC ĐỌC WIREFRAME

```
┌─────────────────────────────────────────────────────┐  ← Khung màn hình
│ HEADER / NAVBAR                                     │  ← Thành phần UI
│─────────────────────────────────────────────────────│  ← Đường phân chia
│                                                     │  ← Vùng nội dung
│  [COMPONENT]  "text"  {VALUE}                     │  ← Component + nội dung
│                                                     │
│  ↓ ↓ ↓ (scroll)                                    │  ← Nội dung cuộn xuống
└─────────────────────────────────────────────────────┘

KÝ HIỆU:
┌────┐  Button / Card
│icon│  Icon
{42}  Dynamic value
"text" Label / placeholder
──    Divider
[___]  Input / Textarea
════════ Progress bar
```

| **Topic Detail** | 1 col lesson list | 2-col lesson grid | 4-col lesson grid + sidebar |

| **Progress** | 2-col stats, vertical charts | 4-col stats, 2-col charts | 4-col stats + sidebar, 2-col charts |
| **Leaderboard** | Compact list, 2-col top-3 | Full list, 4-col | Full list, 4-col + sidebar |
